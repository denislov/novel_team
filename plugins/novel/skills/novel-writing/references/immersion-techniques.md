## 一、代入感的本质

**读者感受的核心**：
- 主角就是"我"
- 这就是"我"身上发生的故事
- 这是一个真实的世界

**深层逻辑**：读者无法与角色共鸣 → 无法代入 → 看不下去。

关键公式：**"我"正在经历什么"事" + 与角色共鸣**

## 二、六大支柱详解

### 1. 基础信息交代与标签化

让读者先认知主角，再认知世界。百字内形成"可想象的空间"：男女、高矮胖瘦、特征。

**示例**：
> "小爷我乃镇南府世子林峰，不过是区区六百两酒钱，醉红楼我欠了三千两都不敢这么对我，还不快给小爷我松绑？"

一句话体现：地点、人物、人物特质、身份地位。谁、在哪、发生了什么。

**原则**：信息简单易消化，让读者快速沉浸在故事事件冲突中。

### 2. 具体化、可视化的熟悉感

**具体化**：描述到读者脑海中能不自觉浮现的东西——被人熟知、可具象的事物。

**可视化**：读者能通过文字想象到日常生活中真实接触过的东西。

**熟悉感**：当文字中出现读者熟悉的场景——大部分读者群体都经历过——他会天然的很熟悉，天然有代入感。

**示例**：
> 夕阳下教学楼后的小树林，少年和少女相对而站。（高考后分手的场景）

这种"接地气"的故事场景，白描风格反而能快速拉近和读者的距离。

### 3. 共鸣（认知共鸣 + 情绪共鸣）

#### 认知共鸣
主角面对选择时，做出的选择与读者重叠——相同的选择、抉择、看法、做法。让书中主角的认知和读者的认知保持同频一致。

#### 情绪共鸣
用亲情等天然纽带作为基础，压制主角，使其陷入愤怒等强情绪。

**常用情绪触发**：

| 场景 | 情绪机制 |
|------|----------|
| 捡废品抚养你长大的奶奶被混混欺负 | 亲情纽带 + 弱势方 |
| 战神的女儿住狗窝 | 激烈强情绪——愤怒 |
| 亲妹妹住院急需手术费，店长克扣工资 | 不平等对待，期望反抗 |

**读者视角**："主角又没错！凭什么？"——以此展开剧情。

### 4. 欲望和好奇心

#### 基础欲望（被动）
每个读者本身存在的欲望：不劳而获、高人一等、权势地位。读者通过小说释放情绪，补充未曾拥有的东西。

#### 主动欲望
作者刻意塑造的"情绪缺口"。如：

> 反派登门作恶，殊不知主角早已今非昔比，读者期待打脸反转。

**制造主动欲望的方法**：
- "憋屈"的对立是"扬眉吐气"
- 压制主角 → 读者期待触底反弹
- 情绪缺口未满足时追更数据好
- 情绪爆发的点要超过读者心理预期

#### 好奇心（期待感）
作者提前构思好的"情绪释放"点。满足读者70%的情绪不够，要超过心理预期。

### 5. 五感代入

依靠读者能具象化的五感描述构建环境氛围：

| 感官 | 示例 |
|------|------|
| 听觉 | 雨水砸在铁皮瓦上的声音 |
| 嗅觉 | 大雨过后泥土的味道 |
| 触觉 | 鱼鳞滑腻的触感 / 潮湿短袖黏在后背 |
| 味觉 | 吃果冻的口感 |
| 视觉 | 夕阳的火烧云 |

**用法**：在大的场景塑造下适当加入一到两种五感描述即可。

### 6. 人设与代入感

**核心**：不要十全十美的人设！

- 人物有缺点 → 经历故事后缺点被填平 → 产生人物弧光
- 微不足道的小缺点让读者觉得平易近人

**示例**：
- 完美漂亮的女角色"李慕婉"，真名叫"牛翠花"
- 温文尔雅霸道总裁回家把衣服丢沙发上，下班路上买淀粉肠

**注意**：缺点暴露的场合很重要，不能在股东大会上吃淀粉肠。

## 三、增强代入感的四种方式

### 1. 熟悉感
熟悉的物品、口头禅、场景。

### 2. 标签
人设标签提前构思好：
- 外在：五官、伤疤、痣、胎记、瞳色、发色、穿着习惯
- 内在：癖好、习惯、缺点、过往经历（人物小传）
- **人设标签前后一致，永不改变，人设崩塌非常可怕**

### 3. 冲突的紧迫性
塑造十分紧迫的故事情节，危机感拉满，读者会被推着往前走。参照灾难电影的手法。

### 4. 接地气
依靠人物缺点和白话描述让读者更容易接收信息，增加内容流畅度。
