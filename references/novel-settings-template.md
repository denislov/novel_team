## 五、禁忌清单

根据作品类型勾选适用项：

- [ ] 禁止圣母心、无脑莽、降智
- [ ] 禁止机械降神（系统不能直接变出超规格物品）
- [ ] 禁止反派降智（对手必须聪明）
- [ ] 禁止流水账（每一行推动剧情或塑造人物）
- [ ] 禁止跪舔外人（利用、压榨、控制）
- [ ] 禁止时间线错乱
- [ ] 禁止主角双标
- [ ] 禁止无脑后宫（女性角色有血有肉）
- [ ] 禁止数据模糊（财务数据精确到个位数）
- [ ] 禁止配角工具人（小弟也有私心和家庭）
- [ ] 禁止文青病（大战前夕不感慨人生）
- [ ] 禁止设定吃书（前后能力范围一致，除非有升级过程）
- [ ] 禁止无铺垫高潮（高潮必须有前戏）
- [ ] 禁止照搬百科（资料融入剧情）
- [ ] 禁止无理由的爱/恨（关系改变必须有事件驱动）
- [ ] 禁止烂尾（每个伏笔都要收回）
- [ ] 禁止不查证写军事（武器数据必须专业）
- [ ] 禁止不懂装懂金融（术语准确，操作合理）
- [ ] 禁止AI式说教
